## Project sinkhole

Project sinkhole is an attempt at creating a small, simple, experimental Operating System.

### Progress

- [ ] Bootloader
- [ ] C++ Support
  - [x] Simple Global Constructor calls
- [ ] Documentation
- [ ] Debugging
- [ ] File System
- [x] Global Descriptor Table
- [x] Interrupts (IDT, IDTR)
- [ ] Keyboard Input
- [ ] Libc
- [ ] Major Drivers
- [x] Memory Detection
- [ ] Memory Map
- [ ] Program Loading
- [ ] Serial Input Output
  - [ ] Input
  - [x] Output
- [x] Stack Smashing Protection
- [x] VGA Output
